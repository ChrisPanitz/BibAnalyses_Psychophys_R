# subfolder terms
Contains the dictionary and different lists of key terms

- compoundTerms_TI_selected: multi-word key terms with at least 5 occurrences across all papers
- dictionary_TI: Excel sheet used to score authors' intent to delete or replace key terms
- keyTerms_TI_raw: list of all extracted key terms (min. 5 occurrences, including multi-word key terms)
- synonyms_TI: list of key terms to be replaced (and of words to replace them with)
- terms2delete_TI: list of key terms to be excluded from analyses
